# Archived Documentation

This folder contains documentation from the previous "Book Heaven" library management system that this template was originally based on.

## Archived Files

The following files have been preserved for reference but are no longer maintained as part of the generic admin template:

- **AI_CHAT.md** - AI-powered book chat system documentation
- **BOOK_CHAT_WITH_AI_PLAN.md** - Implementation plan for AI chat
- **BOOK_CHAT_ZHIPU_AI_PLAN.md** - Zhipu AI integration plan
- **BOOK_CONTENT_EXTRACTION.md** - PDF processing pipeline
- **BOOK_SERIES_IMPLEMENTATION.md** - Book series feature guide
- **BOOK_SUGGESTIONS_PLAN.md** - Book recommendation system planning
- **MOOD_RECOMMENDATIONS.md** - Mood-based recommendation system
- **QUIZ_GAMIFICATION.md** - Quiz and gamification features
- **INFO_TOOLTIP_COMPONENT_PLAN.md** - UI component implementation guide

## Note

These files are kept for historical purposes and may contain useful implementation patterns. However, they are specific to the library management use case and may not directly apply to the generic admin template.

If you're building a library or book-related application, you may find these documents helpful as a reference for implementing similar features.
